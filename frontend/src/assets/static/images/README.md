# Carpeta de Imágenes

Esta carpeta está destinada para almacenar imágenes utilizadas en el sistema de rehabilitación.

## Tipos de imágenes recomendadas:

### Iconos y Logos
- `logo.png` - Logo del sistema
- `favicon.ico` - Icono del navegador
- `icons/` - Carpeta para iconos específicos

### Imágenes de Ejercicios
- `exercises/` - Imágenes relacionadas con ejercicios
- `therapy/` - Imágenes para terapia ocupacional
- `progress/` - Imágenes para indicadores de progreso

### Imágenes de Interfaz
- `ui/` - Elementos de interfaz de usuario
- `backgrounds/` - Fondos y texturas
- `avatars/` - Avatares de usuarios

### Formatos soportados:
- PNG (recomendado para iconos y logos)
- JPG/JPEG (recomendado para fotografías)
- SVG (recomendado para iconos vectoriales)
- WebP (formato moderno con mejor compresión)

### Tamaños recomendados:
- Iconos: 16x16, 32x32, 64x64 píxeles
- Logos: 200x200 píxeles
- Imágenes de ejercicios: 400x300 píxeles
- Fondos: 1920x1080 píxeles

### Optimización:
- Comprimir imágenes antes de subirlas
- Usar formatos apropiados para cada tipo de imagen
- Mantener tamaños razonables para carga rápida
